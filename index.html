Welcome to Zero Merge!

Math Physics Puzzle

1
Red balls = positive numbers, Blue balls = negative numbers. Only the absolute value is shown on each ball. Aim and drop!
2
Balls with the same operator symbol merge on contact and calculate the result.
e.g. 3(×) and 2(×) share ×, so they merge: 3 × (-2) = -6 (blue 6).
3
Calculation order: The ball already in the container (older) becomes the left operand, and the incoming ball (newer) becomes the right operand. The formula is displayed on screen!
4
After merging, the operator evolves in a fixed cycle: + → - → × → ÷ → ^ → +. Plan your drops to create massive chain combos!

Merge & Operator Rules

Balls with the same operator merge! (Older ball = left, Newer ball = right)
After merging, operator evolves: + → - → × → ÷ → ^ → +. Plan your chains!

+
Addition (Add) A + B e.g. Red 3 + Blue 3 = 0 (vanish!)
-
Subtraction (Sub) A - B e.g. Red 3 - Red 3 = 0 (vanish!)
×
Multiplication (Mul) A × B e.g. Blue 2 × Blue 2 = Red 4
÷
Division (Div) A ÷ B e.g. Red 10 ÷ Red 3 = Red 3 (10/3 ≈ 3)
^
Power (Pow) AB e.g. Red 2 ^ Red 3 = Red 8 (23 = 8)
✨ Annihilation: When the result equals 0, the ball vanishes and frees up space!
